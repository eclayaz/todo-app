## ToDo App

Application Feature 
- User registration
- User login/logout 
- Create ToDos
- Mark ToDos as done
- View ToDos
- Filter ToDos
- View Profile
- Upload profile picture

### To install dependencies 
##### `npm install`

### To run the project
##### `npm start`
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

##### Default user login
username: `admin`
password: `admin`

### To run tests
##### `npm test`
